info_outline Autoestudio sobre funciones y procedimientos: Estatutos de repetición.

Realizarás el autoestudio que presenta los conceptos de funciones y procedimientos.

check Objetivos de aprendizaje
  • Conocer los conceptos de función y procedimiento, variable global y local.
  • Diferenciar entre una función y un procedimiento, una variable global y una local.
  • Reconocer la importancia del uso de funciones y procedimiento para el diseño de un programa en C.
list Instrucciones

  • Estudia los siguientes conceptos, haz notas y contesta las preguntas en tu libreta. Te recomendamos el uso de las siguientes fuentes:

    Fernandez, A. Python 3: Al Descubierto Alfaomega, 2013, ISBN 978-607-707-718-3

    Temas / conceptos:

    2. Estructura de una función. (pp. 191)
    3. Funciones de carácter. (pp. 213)
    4. Funciones numéricas. (pp. 215)

    Joyanes Aguilar, L. y Zahonero Martínez, I. Programación en C, C++, Java y UML. México : McGraw Hill, 2014.

    1. Concepto de función. (pp. 177)
    2. Estructura de una función. (pp. 178)
    3. Funciones de carácter. (pp. 199)
    4. Funciones numéricas. (pp. 202)

  • Responde las siguientes preguntas:

    1. 1. ¿Cómo se relacionan las funciones con el la estrategia divide y conquista (divide y vencerás)?
    2. 2. ¿Qué tipo de funciones hay y para qué utilizarías cada una?
    3. 3. ¿Cómo sabes cuando un código debe de ir en una función en lugar de en el main?¿Qué criterios usarías para decidiri esto?
