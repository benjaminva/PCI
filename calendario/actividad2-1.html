Actividad: Autoestudio sobre variables, constantes, tipos de datos y operadores.

info_outline Autoestudio sobre variables, constantes, tipos de datos y operadores: Variables, tipos y operadores.

Realizarás el autoestudio que presenta los conceptos de variable, constante, tipo de datos y operadores.

check Objetivos de aprendizaje
  • Comprender la estructura básica de un programa en Python: variables y su declaración, tipos de datos, los operadores más usados y la prioridad y asociatividad entre ellos.
list Instrucciones

  • Estudia los siguientes conceptos, haz notas y contesta las preguntas en tu libreta. Te recomendamos el uso de las siguientes fuentes:

    Fernandez, A. Python 3: Al Descubierto Alfaomega, 2013, ISBN 978-607-707-718-3

    Temas / conceptos:

    1. Como se ve un programa en Python (pp. 74)
    2. Tipos de datos en python (pp. 76)
    3. Constantes. (pp. 81)
    4. Variables. (pp. 85)
    5. Entradas y salidas. (pp. 89)
    6. Operadores y expresiones. (pp. 101)
    7. Operador de asignación. (pp. 101)
    8. Operadores aritméticos. (pp. 102)
    9. Operadores de incrementación y decrementación. (pp. 105)
    10. Conversión de tipos. (pp. 118)

  • Responde las siguientes preguntas:

    1. Indica en qué orden se evaluaran cada una de la siguientes expresiones:

      9 / 2 * 8 + 5 * 3 - 7 % 4
      7 * 3 - 5 / 2 * 6 + 12
      5 % 2 + 3 % 2 - 4 / 3

    2. Define las constantes necesarias para almacenar la siguiente información:

      El perímetro de un cuadrado cuyo lado mida 4.5.
      El número de ruedas de un triciclo.
      La raíz cuadrada de 2.

    3. Escribe la sentencia en Python que implementa las siguientes expresiones matemáticas:

    1. 1. La raíz cuadrada de B2. + 4AC (A y C son variables distintas).
    2. 2. La raíz cuadrada de X + 4Y3.
    3. 3. La raíz cúbica del producto de X y Y.
    4. 4. El área de un círculo, R2PI.

    4. ¿Para qué sirven los programas o que podrías hacer con ellos?

    5. Da 2 ejemplos de operacione en tu vida diaria que pudieses automatizar en programas, haz una pequeña implementación o algoritmo en pseudocódigo de estas.
